Import Projects from CSV

CSV Format Requirements:

  • name (required): Project name
  • abbreviation (optional): Project abbreviation
  • start_date (optional): Format yyyy-MM-dd
  • end_date (optional): Format yyyy-MM-dd
  • status (required): PLANNING, ONGOING, COMPLETED, CANCELLED
  • team_name (required): Team name (must exist)
  • member_emails (optional): Semicolon-separated emails

Note: Teams and Users must exist before importing projects.
